For more information on HTML DOM, please look at our HTML DOM examples and our HTML DOM reference.

Now You Know HTML DOM, What's Next?

The next step is to learn about PHP or ASP.

While scripts in an HTML file are executed on the client (in the browser), scripts in a PHP/ASP file are executed on the server.

With PHP or ASP you can dynamically edit, change, or add any content of a Web page, respond to data submitted from HTML forms, access any data or databases and return the results to a browser, customize a Web page to make it more useful for individual users.

Since PHP and ASP files are returned as plain HTML, they can be viewed in any browser.

If you want to learn more about PHP, please visit our PHP tutorial.

If you want to learn more about ASP, please visit our ASP tutorial.
