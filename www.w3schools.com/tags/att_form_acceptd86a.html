HTML <form> accept Attribute

HTML <form> tag

Example

Use of the accept attribute in a form (allow two types of files in the file upload - gif and jpeg):

<form action="form_action.asp" accept="image/gif, image/jpeg">
  First name: <input type="text" name="fname" /><br />
  Last name: <input type="text" name="lname" /><br />
  Your image: <input type="file" name="pic" id="pic" /><br />
  <input type="submit" value="Submit" />
</form>

Try it yourself »

Definition and Usage

The accept attribute specifies the type of files that can be submitted through a file upload.

Tip: Avoid using this attribute. File uploads should be validated on the server.

Browser Support

Note: The accept attribute is not properly supported by any of the major browsers.

Syntax

<form accept="value">

Attribute Values

Value Description
MIME_type One or more MIME types that can be submitted/uploaded. To specify more than one MIME type, separate the types with a comma.

Look at IANA MIME types for a complete list of standard MIME types.
