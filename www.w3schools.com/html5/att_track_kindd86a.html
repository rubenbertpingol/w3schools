HTML5 <track> kind Attribute

HTML5 <track> tag

Example

A video element with two subtitle tracks:

<video width="320" height="240" controls="controls">
  <source src="forrest_gump.mp4" type="video/mp4" />
  <source src="forrest_gump.ogg" type="video/ogg" />
  <track kind="subtitles" src="subsnor.srt" srclang="no" label="Norwegian">
  <track kind="subtitles" src="subseng.srt" srclang="en" label="English">
</video>

Definition and Usage

The kind attribute specifies the kind of the track.

Differences Between HTML 4.01 and HTML5

The <track> tag is new in HTML5.

Syntax

<track kind="subtitles" src="norwegiansubs.srt" srclang="no">

Attribute Values

Value Description
captions The track defines captions that will be displayed in the mediaplayer.
chapters The track defines chapters, used to navigate the media resource
descriptions The track defines descriptions, used to describe, with audio, the media's content if the content is not playable/visible
metadata The track defines content used by scripts
subtitles The track defines subtitles, used to display subtitles in a video
