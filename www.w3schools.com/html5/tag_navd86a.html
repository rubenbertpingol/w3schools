HTML5 <nav> Tag

Example

A section of navigation links:

<nav>
<a href="default.asp">Home</a>
<a href="tag_meter.asp">Previous</a>
<a href="tag_noscript.asp">Next</a>
</nav>

Try it yourself »

Browser Support

The <nav> tag is supported in all major browsers.

Definition and Usage

The <nav> tag defines a section of navigation links.

Differences Between HTML 4.01 and HTML5

The <nav> tag is new in HTML5

Tips and Notes

Tip: If you have "previous" and "next" buttons in your document, they should be placed inside the <nav> element.

Global Attributes

The <nav> tag also supports the Global Attributes in HTML5.

Event Attributes

The <nav> tag also supports the Event Attributes in HTML5.
