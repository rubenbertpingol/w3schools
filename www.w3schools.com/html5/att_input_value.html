HTML5 <input> value Attribute

HTML5 <input> tag

Example

A form with initial (default) values:

<form action="demo_form.asp" method="get">
First name: <input type="text" name="fname" value="Peter" /><br />
Last name: <input type="text" name="lname" value="Griffin" /><br />
<input type="submit" value="Submit" />
</form>

Try it yourself »

Definition and Usage

The value attribute specifies a value for an input element.

The value attribute is used differently for different types of input:

  • For types like "button", "reset", "submit" - defines the text on the button
  • For types like "text", "password", "hidden" - defines the initial (default) value of the input field
  • For types like "checkbox", "radio", "image" - defines the value associated with the input. This is the value sent to the form's action URL

Note: The value attribute is required with <input type="checkbox"> and <input type="radio">.

Note: The value attribute cannot be used with <input type="file">.

Differences Between HTML 4.01 and HTML5

None.

Syntax

<input value="value" />

Attribute Values

Value Description
value The value of the input element
