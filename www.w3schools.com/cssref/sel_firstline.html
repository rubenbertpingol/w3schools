CSS :first-line Selector

Complete CSS Selector Reference

Example

Select and style the first line of every <p> element:

p:first-line
{
background-color:yellow;
}

Try it yourself »

Definition and Usage

The :first-line selector is used to add a style to the first line of the specified selector.

Note: The following properties can be used with :first-line: 

  • font properties
  • color properties 
  • background properties
  • word-spacing
  • letter-spacing
  • text-decoration
  • vertical-align
  • text-transform
  • line-height
  • clear

Browser Support

The :first-line selector is supported in all major browsers.
