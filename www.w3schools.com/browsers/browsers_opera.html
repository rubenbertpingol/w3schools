The Opera Browser

The Opera browser is free, and it is the smallest and fastest browser in the world!

Opera started out as a research project in Norway's telecom company, Telenor, in 1994.

The Opera browser is known for its performance, standards compliance and small size.

Opera is Free and Web Standard Compliant!

The Opera browser is free! The earlier banner ads and the licensing fee are removed!!

If your Web site works in Opera, you can be certain that it is standard compliant! Simply write your pages in standards-compliant code, and your site will work in all major browsers and on all major platforms and operating systems.

Opera supports all major Web standards in use, such as CSS, HTML, XHTML, HTTP, DOM, XML, XSL, ECMAScript (JavaScript), PNG, WML, SVG, Unicode, the Unicode Bidirectional Algorithm, and more.

Opera 11.50

Opera 11.50 was released on 28th June, 2011.

New features:

  • Sleeker user interface
  • Added functionality for Opera's pioneering Speed Dial feature - Embed your Sped Dial with extensions that keep you updated instantly instead of thumbnails linking to your favorite links
  • Password Synchronization - Let you synchronize password securely and safely with other Opera browsers
  • Faster graphics performance
  • Improved support for HTML5, CSS3, SVG

Opera 11.50 is available for Linux, Mac, FreeBSD and Windows computers.

Opera 11.10

Opera 11.10 was released in April 2011.

New features:

  • Speed Dial 2.0 - Your favorite pages now have clearer previews, and dials can even dynamically show live content for websites. There is now no limit to how many dials you can use
  • Opera Turbo enhanced - Boost your browsing speed on crowded Wi-Fi hotspots, tethered mobile phones or dial-up connections
  • Simple Plug-in installation - With this release, the most popular plug-in, Adobe Flash Player, installs seamlessly and automatically
  • New CSS3 Support - Opera now supports CSS3 linear gradients and multiple columns

Opera 11.10 is available for Linux, Mac, FreeBSD and Windows computers.

Opera 11

Opera 11 was released in December 2010.

New features:

  • Opera Presto 2.7 rendering engine - Better support for CSS3 and HTML5
  • Tab Stacking - Often we open a lot of tabs and find it difficult to adjust and manage. Opera 11 lets you drag one tab over another to create groups of tabs. Now you can keep dozens of webpages open, organized, and under control
  • Secure address field - An improved address field makes it easier to stay safe on the Web, and it now hides the complexity of long web addresses. It also provides more security information to help you stay safe when browsing; just click the badge for the website to see security information about the site you are visiting
  • Visual Mouse Gestures - Allows common browser actions with a quick flick of your mouse. Holding down the right mouse button on a web page now shows a visual guide that indicates how to perform the available gestures
  • Opera installer for Microsoft Windows platforms - Opera has a new installer written in-house, which is much faster than the previously used MSI installer.
  • New Extensions & Mail Panel - A bunch of new extensions and a new mail panel for a quick usage of email

Opera 11 is available for Linux, Mac, FreeBSD and Windows computers.

Opera 10.6

Opera 10.6 was released in July 2010.

New features:

  • 50 % faster JavaScript engine - over 50% faster than in Opera 10.5
  • Opera Presto 2.6 rendering engine
  • Geolocation services - Opera can prompt you to share your location, making geolocation-supporting websites more useful
  • Offline Web Applications support - Allows documents to communicate with each other regardless of their source domain. Also, does not enable cross-site scripting attacks
  • Web Workers - run scripts in the background, independently of any user interface scripts. This allows long tasks to be executed without yielding to keep the page responsive
  • WebM format - consists of the VP8 video and Vorbis audio codecs. Offers high-quality video with fast seeking. WebM works together with the HTML5 <video> element
  • Search suggestions for selected providers - help you find results more easily

Opera 10.6 is available for Linux, Mac, FreeBSD and Windows computers.

Opera 10.5

Opera 10.5 was released in March 2010.

New features:

  • Better integration with the Windows operating system
  • Better JavaScript engine
  • Opera "O" tab and menu - Replaces the traditional Opera Menu bar and increases vertical space in the Opera desktop window
  • Opera Presto 2.5 rendering engine - Enhanced support for CSS, HTML5 and SVG
  • Opera Vega graphics library - Renders all graphics in Opera
  • Opera Widgets for Desktop - The new Opera Widgets for Desktop is independent from the browser
  • Download acceleration software compatibility - integrates a user's favorite download accelerator with Opera
  • Non-modal notification messages - Notification message windows do no longer interrupt workflow
  • Private browsing - Creating, browsing, and closing private tabs or windows removes all records of Web sites visited through them

Opera 10.5 is available for Linux, Mac, FreeBSD and Windows computers.
