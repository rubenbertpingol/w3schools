PHP asXML() Function

Complete PHP SimpleXML Reference

Definition and Usage

The asXML() function returns the XML document, from a SimpleXMLElement object, as a string.

This function returns FALSE on failure.

Syntax

class SimpleXMLElement
{
string asXML(file)
}

Parameter Description
file Optional. If this parameter is specified, this function writes the XML to a file instead of returning it

Example

XML File

<?xml version="1.0" encoding="ISO-8859-1"?>
<note>
<to>Tove</to>
<from>Jani</from>
<heading>Reminder</heading>
<body>Don't forget me this weekend!</body>
</note>

PHP Code

<?php
if (file_exists('test.xml'))
  {
  $xml = simplexml_load_file('test.xml');
  }

echo $xml->asXML();
?>

The output of the code above will be:

Tove Jani Reminder Don't forget me this weekend!

If you select "View source" in the browser window, you will see the following HTML:

<note>
<to>Tove</to>
<from>Jani</from>
<heading>Reminder</heading>
<body>Don't forget me this weekend!</body>
</note>
