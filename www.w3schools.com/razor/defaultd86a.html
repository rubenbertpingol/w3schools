ASP.NET Razor is a new syntax for writing ASP.NET web pages.

In this tutorial you will learn how to add ASP.NET Razor code to web pages.

ASP.NET Razor is powerfull and easy to learn. You will enjoy it.

Easy Learning with "Run Example"

Our "Run Example" tool makes it easy to learn Razor.

It runs examples and displays the ASP.NET Razor code and the HTML output simultaneously:

Razor Example

<html>
<body>
     <h1>Hello ASP.NET Razor</h1>
     <p>The time is @DateTime.Now</p>
</body>
</html>

Learn by examples!

Because ASP.NET Razor code is executed on the server, you cannot view the code in your browser. You will only see the output from ASP.NET Razor as plain HTML.

At W3Schools every example displays the hidden ASP.NET Razor code. This makes it easier for you to understand how it works.
